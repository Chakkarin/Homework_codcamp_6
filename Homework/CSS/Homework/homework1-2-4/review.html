CSS syntax CSS writing selector combinator

seletor(ตำแหน่งที่เลือกว่าจะกำหนดลักษณะต่าง)

declaration การประกาศลักษณะ ประกอบด้วย

  • property : คุณสมบัติ
  • value : ค่าที่กำหนด

selector { property: value; property : value;}

การเขียนมีอยู่ 3 แบบ

  • Inline CSS : เขียนภายใน tag HTML
  • external CSS : นำเข้าไฟล์ CSS จากภายนอก
    < link rel="stylesheet" type="text/css" href="mystyle.css" >
  • internal CSS : เขียนภายใน Element HTML
    <style>
    ... code CSS ....
    </style>

seletor มี 6 แบบจะประกอบไปด้วย

  • ID
  • Class
  • Pseudo element
  • Pseudo class
  • Combinators
  • attribute selector

[Pseudo classes] จะประกอบไปด้วย

  • :active : เลือกเฉพาะลิงค์ที่ทำงาน
  • :checked : เลือกเฉพาะช่องที่ถูกเช็ค
  • :disbled : เลือกเฉพาะช่องที่ไม่ทำงาน
  • :focus : เลือก input element ที่ถูกคลิก
  • :hover : เลือกเฉพาะบล็อกที่เมาส์เลื่อนโดน
  • :link : เลือกทุกลิงค์ที่มี
  • :required : เลือกเฉพาะช่องที่มี required
  • :target : เลือกเฉพาะ target link

[pseudo element] จะประกอบไปด้วย

  • :: after : เพิ่มข้อความหลังแท็ก
  • :: before : เพิ่มข้อความก่อนแท็ก
  • :: first-letter : เลือกตัวแรกของแท็ก
  • :: first-line : เลือกแถวแรกของแท็ก
  • :: selection : ครอบตัวหนังสือ

[combinators] จะประกอบไปด้วย

  • descendant selector (space) : a b = tag b ทั้งหมดที่อยู่ใน a
  • child selector (>) : a > b = tag b ที่เป็นลูกของ a
  • adjacent sibling selector (+) : a + b = tag b ที่อยู่ล่าง a และติดกับ a เท่านั้น
  • general sibling selector (~) : a ~ b = tag b ที่อยู่ล่าง a ทุก tag

[attribute selector] จะประกอบไปด้วย

  • tag[attribute] : attribute นั้นทั้งหมด
  • tag[attribute = value] : เท่ากับ value ค่านั้น
  • tag[attribute ~=value] : value ที่มีตัวอักษรนี้และไม่ติดกับเครื่องหมายหรือตัวอักษรอื่นๆ
  • tag[attribute |= value] : ขี้นต้นด้วยค่า value นั้นและถูกขั้นด้วย -
  • tag[attribute ^= value] : ขึ้นต้นด้วย ค่า value นั้น
  • tag[attribute $= value] : ลงท้ายด้วย ค่า value นั้น
  • tag[attribute *= value] : value ที่มีตัวอักษรใน value

[combinators] 4 แบบจะประกอบไปด้วย

  • descendant selector (space) : a b = tag b ทั้งหมดที่อยู่ใน a
  • child selector (>) : a > b = tag b ที่เป็นลูกของ a
  • adjacent sibling selector (+) : a + b = tag b ที่อยู่ล่าง a และติดกับ a เท่านั้น
  • general sibling selector (~) : a ~ b = tag b ที่อยู่ล่าง a ทุก tag

selector example


Class

.intro {
background-color: yellow;
}

Id

#intro {
background-color: yellow;
}

pseudo class

selector:pseudo-class {
property: value;
}

Pseudo element

selector::pseudo-element {
property: value;
}

attribute selector

a[target] { // attribute นั้นทั้งหมด เช่น tag a ที่มี target จะเป็นสีเหลือง
background-color: yellow;
}
a[target="_blank"] { // เท่ากับ value ค่านั้น เช่น tag a ที่มี target = _blank จะเป็นสีเหลือง
background-color: yellow;
}
[title~="flower"] { //value ที่มีตัวอักษรนี้และไม่ติดกับเครื่องหมายหรือตัวอักษรอื่นๆ จะมีผลเมื่อมี flower อยู่ใน title
border: 5px solid yellow;
}
[class|=top] { // ขี้นต้นด้วยค่า value ตามด้วย -
background: yellow;
}
[class^="top"] { // ขึ้นต้นด้วย top
background: yellow;
}
[class$="test"] { //ลงท้ายด้วย test
background: yellow;
}
[class*="te"] { //มี te ผสมอยู่
background: yellow;
}

combinator example

div p { // p ทั้งหมดที่เป็นลูก div เท่านั้น
background-color: yellow;
}
div > p { // p ทั้งหมดที่เป็นลูก div ชั้นแรกเท่านั้น
background-color: yellow;
}
div + p { // p ที่อยู่ด้านล่าง tag ปิด div และติดกับ div เท่านั้น
background-color: yellow;
}
div ~ p { // p ที่อยู่ด้านล่าง tag ปิด div ทั้งหมด
background-color: yellow;
}

property มี 6 กลุ่มใหญ่ๆ

  • Property for content
    • color
    • size
    • font
    • text
    • list-style-type
    • Height
    • Width
    • background
  • property for border
    • border
    • table
  • property for padding

  • property for margin

  • property for element
    • display
    • Height
    • Width
    • Position
    • Float
    • Align
    • Flex

value unit มี 2 ประเภท

  • Absolute lengths
    • cm เซนติเมตร
    • mm มิลลิเมตร
    • in นิ้ว
    • px พิกเซล 1/96 นิ้ว
    • pt พอยท์ 1/72 นิ้ว
    • pc พิคาล 1/12 นิ้ว
  • relative lenghts
    • em เท่าขนาดฟอนต์ของ element
    • ex เท่าความสูงของฟอนต์ของ element
    • ch ความกว้างของตัวหนังสือใน parent element
    • rem เท่าขนาดฟอนต์ของ root element
    • pc พิคาล 1/12 นิ้ว
    • vw เปอร์เซ็นต์ความกว้างหน้าจอ
    • vmin เปอร์เซ็นต์ของ view point ต่ำสุด
    • vmax เปอร์เซ็นต์ของ view point สูงสุด
    • % เปอร์เซ็นต์ของ parent element
property เป็นตัวกำหนดว่าต้องการเปลี่ยนแปลงค่าอะไรลงไป ส่วน value คือตัวกำหนดว่าต้องการ ต้องการกำหนด property จำนวนเท่าไหร่

  • inline อยู่บรรทัดเดียวกัน ไม่สามารถปรับแต่ง ความกว้าง ความสูงได้

  • flex การจัดรูปแบบ flex
  • grid การจัดรูปแบบ grid
  • none ซ่อนการแสดงผล
  • hidden ซ่อนการแสดงผลแต่เหลือพื้นที่ไว้
  • grid ต้อง display: grid ใน container ก่อนเสมอ
  • grid layout : คือการกำหนดรูปแบบหน้าจอเป็นลักษณะ คอลัมน์ และ แถว
  • grid column : แถวแนวตั้ง
  • grid row : แถวแนวนอน
  • grid gap : ช่องว่างระหว่าง grid สามารถตั้งค่าได้ 3 แบบ
  • grid column gap : ช่องว่างระหว่างคอลัมน์
  • grid row gap : ช่องว่างระหว่างแถว
  • grid gap : ช่องว่างระหว่างคอลัมน์ และ แถว
  • grid line : ใช้กำหนดการแบ่ง grid ของจำนวนช่องที่ต้องการรวมกันเป็น 1 ช่องใหญ่
  • grid-template-columns : กำหนด ความกว้าง ของแต่ละ คอล้มน์
  • grid-template-rows : กำหนด ความสูง ของแต่ละ แถว
  • gird-line : เส้นแถวหรือเส้นคอลัมน์
  • justify-content: การกำหนดการจัดเรียงแบบแนวนอนของแต่ละแถวของ grid
  • align-content: การกำหนดการจัดเรียงแบบแนวตั้งของแต่ละคอลัมน์ของ grid
  • grid-column: การกำหนดจำนวนช่องของคอลัมน์
  • grid-row: การกำหนดจำนวนช่องของแถว
  • grid-area: การกำหนดพื้นที่ (แถว , คอลัมน์)
  • display: flex ใน container ก่อนเสมอ
  • flex-direction : การกำหนดรูปแบบการจัด item แบบ row หรือ column
  • flex-wrap : กำหนดการม้วนของ item ใน flex-container
  • flex-flow : การผสมระหว่าง flex direction และ flex-wrap
  • justify-content : การจัดการ item แนวนอน justify-content
  • align-items : การจัดการ item แนวตั้ง align-items
  • align-content : การจัดการ flex line align-content
  • order : การจัดอันดับ item ว่า item ไหนมาก่อนหลัง
  • flex-grow : item จะขยายความกว้างได้มากขึ้นเป็นจำนวนเท่า
  • flex-shrink : item จะหดความกว้างได้มากขึ้นเป็นจำนวนเท่า
  • flex-basis : กำหนดความกว้าง item เบื้องต้น
  • flex : shorthand ของ grow, shrink, basis ตามลำดับ
  • align-self : การจัด item แบบเฉพาะบาง item ใน flex-container

justify จะเป็นการจัดรูปแบบในแนวนอน

align จะเป็นการจัดรูปแบบในแนวตั้ง

  • (ใช้ flex-start ใน flex ) (ใช้ start ใน grid) หน้าสุดของ container หรือ ซ้ายสุด
  • (ใช้ flex-end ใน flex ) (ใช้ end ใน grid)ท้ายสุดของ container หรือ ขวาสุด
  • center : ตรงกลาง
  • space-between : พื้นที่ระหว่าง container line(flex) หรือ คอลัมน์(grid) เท่ากัน
  • space-around : พื้นที่รอบๆ(flex) พื้นหน้าและหลัีงคอลัมน์เท่ากัน(grid)
  • space-evenly : พื้นที่หน้าและหลังแต่ละคอลัมน์ เท่ากันหมด (เฉพาะ grid)
