2. differentiation

Tag เหมือน แตกต่าง
hr&br เป็น tag html ที่ช่วยในการแบ่งแยกข้อความ hr จะเป็น tag ที่ขั้นข้อความด้วยการขีดเส้นใต้
br จะเป็น tag ที่ขั้นข้อความด้วยการขึ้นบันทัดใหม่
ul&ol เป็น tag html ที่แสดงรายการเป็นลิส ul แยกลิสเป็นลำดับจุด
ul แยกลิสเป็นลำดับเลข
select&checkbox&radio button เป็น tag html ที่ใช้ในการเลือกข้อมูล select เลือกข้อมูลใน dropdown list
checkbox ใช้เลือกข้อมูลเป็นรายการได้มากกว่า 1
radio ใช้เลือกข้อมมูลการตัดสินใจอย่างใดอย่างหนึ่ง
div&span จัดกลุ่มข้อความที่อยู่ภายใต้ tag div จัดกลุ่มข้อความ
span จัดกลุ่มข้อความที่อยู่ภายใต้ tag อื่นๆ
script&style ใช้ สคิปอื่นเข้ามาจัดการ HTML script ใช้ javascript เข้ามาจัดการการทำงาานบน HTML
style ใช้ css เพิ่มเติมในส่วน HTML
